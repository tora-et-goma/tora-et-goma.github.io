Web Designer Yoshiko Watanabe

フランス在住のWebデザイナー、Yoshiko Watanabeです。オンラインスクールでWeb制作について学び、サービス業界から憧れていたWebデザイナーへと転職活動中です。これからさらにWeb制作の技術や知識を身につけて、Webサイト運用や問題解決、Web周りの事務作業効率化アドバイスもできるデザイナーになりたいと思っています。

Skills

Web

HTML/CSSでのコーディングができます。JavaScript、Jquery、PHP、WordPressも現在勉強中です。

EUC

Webコーディングのほかに、エクセル、アクセスが得意です。ワードプレスとアクセスを活用し、社内データをワードプレスへアップする作業などを効率化いたします。作業報告をエクセル等で自動送付など、EUCを活用し、日々の業務を効率化いたします。

English

言語は日本語、英語、フランス語で対応が可能です。海外在住ですが連絡はさまざまなツールを活用し、日本の日中にとることが出来ます。
